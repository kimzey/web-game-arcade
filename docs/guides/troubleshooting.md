# คู่มือการแก้ไขปัญหาทั่วไป (Troubleshooting Guide)

## ปัญหาที่พบบ่อยและวิธีแก้ไข

### 1. ปัญหาการเชื่อมต่อ P2P

#### ปัญหา: ไม่สามารถสร้างห้องได้
**อาการ**: กดปุ่ม "สร้างห้อง" แล้วไม่มีอะไรเกิดขึ้น หรือแสดงข้อความ error

**วิธีแก้ไข**:
1. ตรวจสอบว่าเบราว์เซอร์รองรับ WebRTC:
   ```javascript
   console.log('WebRTC supported:', !!window.RTCPeerConnection)
   ```
2. ตรวจสอบ console สำหรับ error messages
3. ลองรีเฟรชหน้าและลองใหม่
4. ลองใช้เบราว์เซอร์อื่น (แนะนำ Chrome หรือ Firefox)

#### ปัญหา: ไม่สามารถเข้าร่วมห้องได้
**อาการ**: ใส่รหัสห้องแล้วแสดงข้อความ "รหัสไม่ถูกต้อง" หรือการเชื่อมต่อล้มเหลว

**วิธีแก้ไข**:
1. ตรวจสอบว่าคัดลอกรหัสมาครบถ้วนและไม่มีช่องว่างเพิ่มเติม
2. ตรวจสอบว่ารหัสยังไม่หมดอายุ (ถ้ามีการตั้งเวลาหมดอายุ)
3. ลองคัดลอกรหัสใหม่แล้วลองใหม่
4. ตรวจสอบว่าอยู่ในเครือข่ายเดียวกับผู้สร้างห้อง

#### ปัญหา: การเชื่อมต่อขาดหายระหว่างเกม
**อาการ**: เล่นเกมอยู่ดีๆ แล้วเกมค้างหรือข้อมูลไม่ส่งถึงกัน

**วิธีแก้ไข**:
1. รอสักครู่เพื่อดูว่าการเชื่อมต่อจะกลับมาเองหรือไม่
2. ลองรีเฟรชหน้าและเข้าร่วมห้องใหม่
3. ตรวจสอบสถานะเครือข่ายของคุณ
4. ถ้าเป็นปัญหาบ่อยๆ ลองใช้เครือข่ายอื่น

### 2. ปัญหา UI/UX บนอุปกรณ์มือถือ

#### ปัญหา: ปุ่มเล็กเกินไป กดยาก
**อาการ**: ปุ่มต่างๆ เล็กมาก กดได้ยากบนหน้าจอสัมผัส

**วิธีแก้ไข**:
1. ใช้มุมมองแนวนอน (landscape mode) แทนแนวตั้ง
2. ซูมหน้าเว็บ (pinch-to-zoom) เพื่อขยายปุ่ม
3. ใช้เบราว์เซอร์โหมดเดสก์ท็อป (desktop mode) ถ้ามี
4. รอให้มีการอัปเดตที่ปรับปรุง UI สำหรับมือถือ

#### ปัญหา: ไม่มีปุ่มควบคุมเกมบนมือถือ
**อาการ**: ไม่มีปุ่มควบคุมเกม เล่นได้ยากบนอุปกรณ์มือถือ

**วิธีแก้ไข**:
1. ใช้เบราว์เซอร์ที่มีคีย์บอร์ดเสมือน (virtual keyboard)
2. รอการอัปเดตที่เพิ่มปุ่มควบคุมแบบสัมผัส
3. เชื่อมต่อคีย์บอร์ดภายนอก (Bluetooth/USB) ถ้าอุปกรณ์รองรับ

#### ปัญหา: ไม่มีปุ่มเริ่มใหม่เมื่อเกมจบ
**อาการ**: เมื่อเกมจบ ไม่มีปุ่มเริ่มใหม่บนอุปกรณ์มือถือ

**วิธีแก้ไข**:
1. รีเฟรชหน้าเพื่อเริ่มเกมใหม่
2. กลับไปที่หน้าเลือกเกมและเลือกเกมเดิม
3. รอการอัปเดตที่เพิ่มปุ่มเริ่มใหม่สำหรับอุปกรณ์สัมผัส

### 3. ปัญหาประสิทธิภาพ

#### ปัญหา: เกมกระตุกหรือช้า
**อาการ**: เกมทำงานช้า ภาพกระตุก หรือตอบสนองช้า

**วิธีแก้ไข**:
1. ปิดแท็บหรือแอปพลิเคชันอื่นๆ ที่ใช้ทรัพยากร
2. ตรวจสอบว่าอุปกรณ์มีประสิทธิภาพเพียงพอ
3. ลองลดคุณภาพกราฟิก (ถ้ามีตัวเลือก)
4. อัปเดตเบราว์เซอร์เป็นเวอร์ชันล่าสุด
5. รีสตาร์ทอุปกรณ์

#### ปัญหา: ใช้งานแบตเตอรี่เร็ว
**อาการ**: แบตเตอรี่หมดเร็วเวลาเล่นเกม

**วิธีแก้ไข**:
1. ลดความสว่างหน้าจอ
2. ปิดฟีเจอร์ที่ไม่จำเป็น เช่น Bluetooth, GPS
3. ใช้โหมดประหยัดแบตเตอรี่ (ถ้ามี)
4. ลองเล่นเกมขณะชาร์จ

### 4. ปัญหาเบราว์เซอร์เฉพาะ

#### ปัญหา: ไม่ทำงานบน Safari
**อาการ**: ฟีเจอร์บางอย่างไม่ทำงานบน Safari บน iOS/macOS

**วิธีแก้ไข**:
1. อัปเดต Safari เป็นเวอร์ชันล่าสุด
2. อนุญาตการใช้ WebRTC ในการตั้งค่า Safari
3. ลองใช้เบราว์เซอร์อื่นเช่น Chrome บน iOS
4. รอการอัปเดตที่รองรับ Safari ดีขึ้น

#### ปัญหา: ไม่ทำงานบน Internet Explorer
**อาการ**: ไม่สามารถใช้งานบน Internet Explorer

**วิธีแก้ไข**:
1. IE ไม่รองรับ WebRTC ต้องใช้เบราว์เซอร์สมัยใหม่
2. แนะนำให้ติดตั้ง Chrome, Firefox, Edge หรือ Safari

## การรายงานปัญหา

### ข้อมูลที่ต้องรวมเมื่อรายงานปัญหา
1. ประเภทอุปกรณ์ (Desktop/Mobile/Tablet)
2. ระบบปฏิบัติการและเวอร์ชัน
3. เบราว์เซอร์และเวอร์ชัน
4. ข้อความ error (ถ้ามี)
5. ขั้นตอนการทำซ้ำปัญหา
6. ภาพหน้าจอหรือวิดีโอ (ถ้าเป็นไปได้)

### รายงานปัญหาผ่าน:
- GitHub Issues
- อีเมลสำหรับรายงานปัญหา
- ช่องทางสื่อสารของทีมงาน

## เคล็ดลับการใช้งาน

### สำหรับผู้เล่น Desktop
1. ใช้เบราว์เซอร์ Chrome หรือ Firefox เพื่อประสิทธิภาพที่ดีที่สุด
2. ปิดแท็บที่ไม่ใช้เพื่อประหยัดทรัพยากร
3. ใช้การเชื่อมต่ออินเทอร์เน็ตที่เสถียร

### สำหรับผู้เล่น Mobile
1. ใช้มุมมองแนวนอน (landscape) สำหรับประสบการณ์ที่ดีกว่า
2. เชื่อมต่อ WiFi เพื่อประสิทธิภาพที่ดีกว่า 4G/5G
3. ชาร์จอุปกรณ์ขณะเล่นเพื่อป้องกันแบตเตอรี่หมด

### สำหรับการเชื่อมต่อ
1. ตรวจสอบว่ารหัสห้องคัดลอกมาครบถ้วน
2. ส่งรหัสผ่านช่องทางที่เชื่อถือได้ (chat, email)
3. รอให้การเชื่อมต่อสำเร็จก่อนเริ่มเกม

## Definition of Done (DoD)

- ✅ ระบุปัญหาที่พบบ่อยและวิธีแก้ไข
- ✅ มีข้อมูลเพียงพอสำหรับผู้ใช้ในการแก้ไขปัญหาด้วยตนเอง
- ✅ ระบุข้อมูลที่ต้องการเมื่อรายงานปัญหา
- ✅ มีเคล็ดลับสำหรับการใช้งานแต่ละแพลตฟอร์ม
- ✅ มีข้อมูลที่ช่วยในการป้องกันปัญหา

## Acceptance Criteria

- ✅ ผู้ใช้สามารถใช้คู่มือนี้แก้ไขปัญหาที่พบบ่อยได้
- ✅ มีข้อมูลเพียงพอสำหรับการรายงานปัญหาที่มีประโยชน์
- ✅ ช่วยลดภาระของทีมสนับสนุนด้านเทคนิค
- ✅ สามารถนำไปอัปเดตเมื่อมีปัญหาใหม่ๆ ที่พบ
- ✅ มีข้อมูลที่เข้าใจง่ายแม้สำหรับผู้ใช้ที่ไม่ใช่เทคนิค
